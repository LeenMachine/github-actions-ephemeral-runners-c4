# Diagram Images

This directory contains static exports of architecture diagrams.

## Contents

- **System Context Diagrams**: High-level view showing actors and external systems
- **Container Diagrams**: Major components and their relationships
- **Component Diagrams**: Internal structure of containers
- **Sequence Diagrams**: Workflow and process flows
- **Network Diagrams**: Cilium network policies and VPC layout

## Formats

Diagrams are provided in:
- **PNG**: For embedding in presentations and wikis
- **SVG**: For high-quality scaling and web use
- **Source Files**: Draw.io, Mermaid, PlantUML sources for editing

## Usage

These diagrams are referenced throughout the documentation in `/docs/architecture/`.

To regenerate diagrams from source:
1. Edit source files (`.drawio`, `.mermaid`, `.puml`)
2. Export to PNG/SVG
3. Update references in markdown files
